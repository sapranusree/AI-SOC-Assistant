# AI SOC Assistant

AI-powered Security Operations Center (SOC) Assistant built using Python, Streamlit, and Gemini AI.

## Features
- Log Analysis
- IOC Extraction
- Threat Detection
- AI-Powered Recommendations
- Incident Report Generation

## Technologies
- Python
- Streamlit
- Gemini AI
- ReportLab
- dotenv
